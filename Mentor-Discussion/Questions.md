### 정해성

- JavaScript에서 `==`과 `===`의 차이는 알겠는데, 실제로 어떤 상황에서 `==`을 써야 하는지 감이 잘 안 잡혀요. 실무에서는 `==`을 아예 안 쓰는지 궁금합니다

```js
console.log(0 == '0') // true
console.log(0 === '0') // false
```
